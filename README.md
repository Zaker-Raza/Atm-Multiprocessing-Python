# Atm-Multiprocessing-Python
ATM system using python multiprocessing ( lock &amp; value)

This project simulates an ATM system with:
  -check balance
  -Deposit
  -Withdraw
  -Shraed balance using value
  -Lock for synchronization 

Concepts used:
  -Multiprocessing
  -LOCk
  -Value

How to run:
python atm-p1.py
